We didn't just build a shell. We aligned the core stack to guarantee the ultimate desktop experience.

Built atop Debian 13 "Trixie" testing base. Experience absolute security and access the exact same package repository. Rock-solid stability out of the box.

Fully Auditable & Open

No hidden data collection, no telemetry. Read the source, audit the packages, compile your own components. The system belongs completely to you.

Adapted Down to the Core

From the system shell and file managers down to custom browsers and panels. Every element has been fine-tuned to mirror the visual behavior of the target OS.
